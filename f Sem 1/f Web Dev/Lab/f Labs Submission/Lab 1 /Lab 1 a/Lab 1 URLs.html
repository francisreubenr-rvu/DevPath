Absolute vs Relative URLs

Absolute URLs

Include the full web address from the beginning.

Example: https://www.example.com/images/photo.jpg

Visit Google (absolute link)

Relative URLs

Relative to the current page location.

Example: page2.html or images/photo.jpg

Go to About page (relative link)

Type Example
Absolute https://example.com/page.html
Relative (same folder) page2.html
Relative (subfolder) images/photo.jpg
Relative (parent folder) ../index.html
