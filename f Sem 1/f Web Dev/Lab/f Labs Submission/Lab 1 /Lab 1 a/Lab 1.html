What is a Website?

A website is a collection of pages on the internet that you can visit using a web browser.

About URLs

A URL is the web address you type into your browser, like www.example.com

URLs can be absolute (complete address) or relative (just the page name).

Absolute URL Example:

https://www.google.com/search

Relative URL Example:

page2.html
